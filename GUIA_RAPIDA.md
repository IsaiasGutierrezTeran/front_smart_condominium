# 🚀 Guía Rápida - Smart Condominium

## Para Usuarios que Solo Quieren Ejecutar el Proyecto

### Pasos Mínimos:

1. **Descargar el proyecto:**
   ```bash
   git clone https://github.com/IsaiasGutierrezTeran/front_smart_condominium.git
   cd front_smart_condominium
   ```

2. **Instalar dependencias:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

4. **Abrir en el navegador:**
   ```
   http://localhost:5173
   ```

### Si tienes problemas:

**Error de Node.js:**
- Descargar e instalar Node.js desde: https://nodejs.org
- Usar versión 18 o superior

**Error de Git:**
- Descargar e instalar Git desde: https://git-scm.com
- O descargar ZIP desde GitHub directamente

**Error de dependencias:**
```bash
rm -rf node_modules
npm install --legacy-peer-deps --force
```
